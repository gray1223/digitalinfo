    About

    Overview

         This website was created for the Segment 2 Collaboration as a demonstration of my HTML/CSS knowledge. I designed this website in a similar manner to most projects I complete. As a designer, I have always liked the colors #282828 and #555555, both of which are on this page. To make this project a collarboration, my Dad sat next to me and I explained what different HTML and CSS elements do, and recieved agreement on color and design choices. I also published adaptations for < 480px, < 768px, < 1024px, and < 1440px device screen widths.

    Process

         I used the Atom code editor for this project, and git versioning to upload and refine my code. My go-to choice for positioning has always been CSS Flexbox (rather than CSS grid), and I don't like Bootstrap as much as using my own designs and code. As a developer, I have completed a handful of other website designs. They are avaliable in the next section, with links to their live copies and github repositories.
